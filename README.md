# Real-Time-Object-Detection-using-YOLOv8-and-Streamlit
Real-Time Object Detection project using YOLOv8 and Streamlit. The model detects objects in images, videos, and webcam feed with live confidence scores. Includes a simple web UI, pre-trained models, and complete code for running and testing object detection.
Running Process (Compact Version)

Install dependencies: pip install -r requirements.txt

Run the Streamlit app: streamlit run app.py

Choose image, video, or webcam in the UI

Select YOLO model and confidence

View real-time detection results on screen
